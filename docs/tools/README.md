# 常用工具

## ADB

Android 调试桥 (adb) 是一种功能多样的命令行工具，可让您与设备进行通信。adb 命令可用于执行各种设备操作，例如安装和调试应用。

[官方资料](https://developer.android.google.cn/studio/command-line/adb?hl=zh_cn)
